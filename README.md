# MyBuddy
GenAI based service to create personal buddy per project


## Local Setup

``python3 -m venv venv``

``source venv/bin/activate``

``pip install -r requirements.txt``


## Create local env file .env with following variables

IMS_ORG_ID=\
IMS_CLIENT_ID=\
IMS_CLIENT_SECRET=\
IMS_AUTH_CODE=\


## Run

```python src/main.py``
